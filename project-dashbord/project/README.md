# TeenEase Dashboard

A professional student dashboard for teenagers to book career counselling, guidance sessions, mentorship, and tuition classes.

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Recharts
- Lucide React

## Features

- Authentication (Login, Register, Forgot Password)
- Student Dashboard with stats and charts
- Career Counselling Booking
- Guidance Session Booking
- Tuition Class Booking
- Appointment Calendar
- Booking History
- Notifications
- Profile Editing
- Settings (General, Notifications, Security)
- Help & Support
- Admin Panel (Students, Counsellors, Bookings, Reports)
- Dark/Light Mode
- Responsive Design

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Ready for Vercel deployment. The `dist` folder is the build output.
